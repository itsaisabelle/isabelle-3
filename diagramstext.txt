LINK [M20] L4 -> L2 | RETURN      | cancellationConfirmed

LINK [M21] L3 -> L4 | SYNCHRONOUS | rsvpToEvent(studentB, eventID)
LINK [M22] L4 -> L5 | SYNCHRONOUS | checkCapacity(eventID)
LINK [M23] L5 -> L4 | RETURN      | capacityOK
LINK [M24] L4 -> L3 | RETURN      | rsvpConfirmed

--- CONSTRAINTS ---
CONSTRAINT: M13-M16 only occurs when currentRsvpCount >= maxCapacity
CONSTRAINT: M21-M24 only possible after a cancellation frees a slot (M17-M20)
